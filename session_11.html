🔁 السر

السر: نحطها جوه forever + if

البلوكات دي لوحدها مبتعملش حاجة… لازم نسألها طول الوقت! عشان كده بنحطها جوه forever (لوب لا نهائي) وجواها if (الشرط). كده البرنامج بيفضل يراقب باستمرار 👀 لحد ما اللمس يحصل:

forever
  if <touching [edge] ?> then
    say "اصطدمت!"
✅ ترجمة بالعربي: "طول الوقت… لو الشخصية لمست الحافة → قول اصطدمت!". لو شلنا forever، البرنامج هيشيك مرة واحدة بس وخلاص — وده مش اللي عايزينه!

🎮 جرّب بنفسك: الفأر، الحيطة، والهدف!

ده فأر 🐭 بتتحكم فيه بالأسهم تحت ⬆️⬇️⬅️➡️. حاول توصله للمربع الأخضر 🟢 (الهدف) من غير ما تلمس الحيطة الحمرا 🔴. البرنامج بيراقب باستمرار — زي ما عملنا بالظبط بـ forever + if!

🎯
🐭
🕹️ استخدم الأسهم تحرّك الفأر. وصّله للهدف الأخضر! (تقدر كمان تستخدم أسهم الكيبورد)

📝 تمارين صغيرة (فكّر الأول!)

1️⃣ لو الشخصية لمست الحافة تقول "آخ!"

عايزين كل ما الشخصية تلمس حافة المسرح، تقول "آخ!" لمدة ثانية. أنهي بلوكات تجمّع؟

👁️ شوف الإجابة
forever
  if <touching [edge] ?> then
    say "آخ!" for 1 seconds

2️⃣ لو لمست الشخصية التانية تعمل صوت

لو الشخصية بتاعتنا لمست Sprite2، عايزينها تشغّل صوت "Meow". أنهي شرط وأنهي بلوك صوت؟

👁️ شوف الإجابة
if <touching [Sprite2] ?> then
  start sound [Meow]

طبعاً بنحطها جوه forever عشان تفضل تراقب 😉

3️⃣ سؤال: ليه بنحط الـ if جوه forever؟

فكّر كويس… ليه مش بنكتب الشرط لوحده بس؟ ليه لازم يكون جوه لوب لا نهائي؟

👁️ شوف الإجابة
عشان البرنامج يفضل يراقب اللمس طول الوقت مش مرة واحدة! 👀 لو كتبنا الشرط لوحده، البرنامج هيشيك مرة واحدة في أول ما يشتغل وخلاص. لكن جوه forever، هيشيك… ويشيك… ويشيك للأبد، فأول ما يحصل لمس يحسّ بيه على طول! ⚡

🚧 المشروع: لا تلمس الحيطة!

دلوقتي دورك! هنعمل لعبة صغيرة: شخصية بتمشي بأسهم الكيبورد، لازم توصل للون الأخضر 🟢 (الفوز) من غير ما تلمس اللون الأحمر 🔴 (الحيطة). لو لمست الحيطة → ترجع من الأول! ده هيجهّزنا للعبة المتاهة الكبيرة بعدين 🧩

🎯 الهدف: حرّك الشخصية بالأسهم، اوصل للأخضر تكسب 🏆، والمس الأحمر ترجع تبدأ من الأول 😅

📋 الخطوات (نفّذها في سكراتش):

1
ارسم خلفية (backdrop) فيها حيطة حمرا 🔴 و مربع أخضر 🟢 للهدف. وحط الشخصية في نقطة البداية.
2
ابدأ بحدث وخلّي الشخصية ترجع لنقطة البداية:
when 🚩 clicked go to x: 0 y: 0
3
خلّي الشخصية تتحرك بالأسهم: لكل سهم حدث "key pressed" يحرّكها في اتجاهه:
when up arrow key pressed point in direction 0 move 10 steps

(اعمل نفس الفكرة للأسهم التانية: يمين 90، شمال احنا بنقول 90 ناحية الشمال، تحت 180)

4
حط لوب لا نهائي يراقب اللمس باستمرار:
forever if touching color 🔴 ? then
5
لو لمس الأحمر → يرجع البداية ويقول "حاول تاني!":
say حاول تاني! 😅 for 1 sec go to x: 0 y: 0
6
وكمان جوه نفس الـ forever، شيك على الهدف: لو لمس الأخضر → "فزت!":
if touching color 🟢 ? then say فزت! 🏆

🧩 الكود كامل (شكله في سكراتش):

when 🚩 clicked
go to x: 0 y: 0
forever
  if <touching color [🔴] ?> then
    say "حاول تاني! 😅" for 1 seconds
    go to x: 0 y: 0
  if <touching color [🟢] ?> then
    say "فزت! 🏆"

when [up arrow] key pressed
  point in direction 0
  move 10 steps
🌟 تحديات إضافية (لو خلصت بسرعة):
• ضيف حيطة حمرا تانية في مكان مختلف عشان اللعبة تصعب 🧱
• خلّي الشخصية تعمل صوت لما تلمس الحيطة 🔊
• ضيف عدّاد للمحاولات (هنتعلمه قريب!)

✅ قائمة المراجعة (دوس على اللي خلّصته):

  • عملت خلفية فيها حيطة حمرا وهدف أخضر
  • الشخصية بترجع نقطة البداية في الأول
  • الشخصية بتتحرك بأسهم الكيبورد
  • استخدمت لوب forever يراقب باستمرار
  • لو لمست الأحمر بترجع وتقول "حاول تاني!"
  • لو لمست الأخضر بتقول "فزت!"
