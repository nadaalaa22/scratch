📝 الخوارزمية (Algorithm) - آلة حاسبة بسيطة

الخوارزمية هي مجموعة خطوات مرتبة بالترتيب عشان نحل مشكلة. خلينا نشوف خطوات عمل آلة حاسبة لجمع رقمين:

  • 1 Start / البداية: اضغط على العلم الأخضر 🏁 (when green flag clicked)
  • 2 Ask for first number: استخدم بلوك ask [Enter first number] and wait (من فئة Sensing) عشان نخلي المستخدم يدخل الرقم الأول.
  • 3 Store in variable: خزن القيمة اللي دخلها المستخدم في متغير اسمه num1 باستخدام بلوك set [num1] to (answer) (من فئة Variables).
  • 4 Ask for second number: كرر الخطوة 2 عشان تاخد الرقم الثاني، وخزنه في متغير num2.
  • 5 Calculate / المعالجة: استخدم بلوك العمليات الحسابية ( + ) من فئة Operators (لونها أخضر) لحساب num1 + num2 وخزن الناتج في متغير result.
    set [result] to ((num1) + (num2))
  • 6 Output / المخرجات: استخدم بلوك say (join [The sum is: ] (result)) for 2 seconds (من فئة Looks) عشان تعرض النتيجة على المسرح.
  • 7 End / النهاية: البرنامج يخلص والقطة تقول النتيجة 🎉
💡 ملاحظة مهمة: لو غيرنا ترتيب الخطوات (مثلاً حسبنا قبل ما نخزن الأرقام) مش هتشتغل الآلة الحاسبة. الخوارزمية محتاجة ترتيب دقيق!

📦 Variables (المتغيرات) – صناديق التخزين في البرمجة

المتغير (Variable) زي صندوق صغير بنحط فيه قيمة مؤقتة (رقم، نص، إلخ) عشان نستخدمها بعدين. كل متغير ليه اسم و قيمة.

إزاي نعمل متغير في سكراتش؟

  1. روح على قسم Variables (لونه برتقالي).
  2. اضغط على Make a Variable.
  3. اكتب اسم المتغير (مثلاً: num1، num2، result).
  4. اختار For all sprites عشان كل الكائنات تشوفه، أو For this sprite only عشان يبقى خاص بكائن واحد.
  5. اضغط OK، هتلاقي بلوكات جديدة ظهرت:
set [num1 v] to (0) // يخزن قيمة جديدة في المتغير
change [num1 v] by (1) // يزيد أو ينقص قيمة المتغير
show variable [num1 v] // يظهر قيمة المتغير على المسرح
hide variable [num1 v] // يخفي قيمة المتغير

📌 تخزين القيم (Storage): بنستخدم set [variable] to (value) عشان نحط قيمة معينة. والقيمة ممكن تكون:

  • رقم ثابت: set [num1] to (5)
  • إجابة المستخدم: set [num1] to (answer) (بعد بلوك ask)
  • ناتج عملية حسابية: set [result] to ((num1) + (num2))
  • نص: set [name] to [Ahmed]
🧠 تذكر: المتغيرات بتخزن قيمة واحدة بس في المرة الواحدة. لو خزنت قيمة جديدة، القيمة القديمة بتتمسح. زي ما تحط ورقة جديدة في صندوق، الورقة القديمة تروح.

🧮 Operators – العمليات الحسابية في Scratch

بلوكات Operators (لونها أخضر فاتح 🟢) هي اللي بتخلينا ننفذ العمليات الحسابية.

Addition (+)

(number1) + (number2)

(5) + (3)

النتيجة: 8

Subtraction (-)

(number1) - (number2)

(10) - (4)

النتيجة: 6

Multiplication (*)

(number1) * (number2)

(7) * (6)

النتيجة: 42

Division (/)

(number1) / (number2)

(20) / (4)

النتيجة: 5

💻 ملاحظة: في سكراتش، علامة الضرب هي * وعلامة القسمة هي /. وممكن تستخدم المتغيرات جوه البلوكات دي، مثال: ((num1) + (num2))

🎮 Important Scratch Blocks (Ask, Set, When Sprite Clicked)

📢 ask [ ] and wait

من فئة Sensing (لونه أزرق فاتح). بيعرض سؤال للمستخدم ويستنى يكتب إجابته.

ask [Enter first number:] and wait

الإجابة بتتخزن في بلوك answer

مثال: بعد ما المستخدم يكتب رقم، نقدر نخزنه في متغير: set [num1] to (answer)

📦 set [variable] to [ ]

من فئة Variables (لونه برتقالي). بيخزن قيمة جديدة في المتغير.

set [num1] to (5)
set [num2] to (answer)
set [result] to ((num1) + (num2))

🐱 when this sprite clicked

من فئة Events (لونه أصفر). بينفذ الأوامر اللي تحته لما المستخدم يضغط على الكائن (Sprite).

when this sprite clicked

مثلاً: نضغط على شكل علامة "+" فتشتغل عملية الجمع.

🎨 How to Draw a Custom Sprite (كائن مخصص)

هنرسم 4 كائنات لكل عملية حسابية (➕➖✖️➗) ونستخدم بلوك when this sprite clicked عشان ننفذ العملية.

📌 خطوات الرسم في Scratch:

  1. افتح Scratch Editor.
  2. في منطقة Sprites، اضغط على Choose a Sprite → Paint (أو أيقونة الفرشاة).
  3. استخدم أداة Text (T) عشان تكتب علامة + أو - أو × أو ÷.
  4. غير اللون والخط عشان الشكل يبان جميل.
  5. امسح الخلفية (اجعلها شفافة) عشان الكائن يظهر من غير مربع أبيض.
  6. سمي الكائن باسم مناسب (مثل "AddButton", "SubtractButton").
  7. كرر الخطوات لبقية العمليات.

➕ Add

ارسم دائرة خضراء و +

➖ Subtract

ارسم دائرة حمراء و -

✖️ Multiply

ارسم دائرة برتقالية و ×

➗ Divide

ارسم دائرة بنفسجية و ÷
