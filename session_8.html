🏆 الاختبار العملي: صديقي القط المتفاعل 🐱

وقت تثبت إنك بقيت تعرف تبرمج! اعمل المشروع ده بنفسك في سكراتش

📋 قبل ما تبدأ — اقرأ ده

النهارده مفيش شرح جديد خالص! ده اختبارك العملي على كل اللي اتعلمناه من Session 1 لـ 6. هتبني مشروع واحد ممتع اسمه "القط المتفاعل" 🐱 — قط بيرحّب بيك باسمك، بيتحرّك ويبدّل شكله، بيعمل صوت لما تدوس عليه، وبيردّ عليك حسب إجابتك.

🎯 المشروع مقسّم لـ 5 مهام. كل مهمة بتختبر فكرة من سيشن. اعملهم بالترتيب وهتلاقي القط اتكوّن لوحده!

اللي محتاجه: افتح سكراتش، سيب القطة 🐱 الموجودة (Sprite1)، وابدأ تركّب البلوكات زي ما هنقولك في كل مهمة 👇

when 🚩 clicked move forever say start sound if / else ask

🎬 شكل المشروع لما يخلّص (معاينة)

كده هيبقى شكل القط بتاعك: بيتحرك، بيبدّل شكله (أنيميشن)، يرتد عند الحافة، ولما تدوس عليه يقولك "هاي!".

🐱

(دوس على القط 🐱 وهو بيتحرك عشان يقولك هاي! — دي معاينة بس، انت اللي هتبنيها)

1

القط يرحّب بيك باسمك

Session 1 + 6: أحداث + سؤال

خلّي البرنامج يسألك عن اسمك أول ما تضغط العلم الأخضر، وبعدين القط يرحّب بيك باسمك.

when 🚩 clicked
ask "إيه اسمك؟" and wait
say (join "أهلاً يا " (answer)) for 2 seconds
💡 تلميح: بلوك join من قسم Operators بيلزّق كلمتين مع بعض (الترحيب + الاسم).
2

القط يتحرّك ويبدّل شكله للأبد

Session 2 + 3 + 4: حركة + لوب + مظاهر

خلّي القط يتحرك على المسرح، يرتد لما يوصل الحافة، ويبدّل أزياءه عشان يبان بيتحرّك (أنيميشن) — وكل ده طول الوقت بلوب.

when 🚩 clicked
set rotation style [left-right]
forever
  move 10 steps
  if on edge, bounce
  next costume
  wait 0.3 seconds
💡 تلميح: set rotation style left-right بتمنع القط إنه يمشي مقلوب لما يرتد.
3

القط يعمل صوت لما تدوس عليه

Session 5: الصوت

لما تدوس على القط، يعمل صوت "مياو" ويقولك "هاي!".

when this sprite clicked
  start sound [Meow]
  say "هاي! 😺" for 1 seconds
4

القط ياخد قرار (سؤال نعم/لأ)

Session 6: الشرط if / else

خلّي القط يسألك سؤال، ولو إجابتك "نعم" يعمل حاجة، ولو لأ يعمل حاجة تانية. ده قلب الاختبار! ⚡

when 🚩 clicked
ask "تحب أكبر شوية؟ اكتب نعم" and wait
if <answer = "نعم"> then
  change size by 30
  say "شكراً! بقيت أكبر 😻" for 2 seconds
else
  say "تمام، هفضل زي ما أنا 🙂" for 2 seconds
💡 تلميح: استخدم بلوك المقارنة answer = نعم من قسم Operators جوه الـ if.
5

زرار سحري بالكيبورد

Session 3 + 4: حدث + مظاهر

لما تضغط زرار المسافة (space) القط يكبر شوية، ولما تضغط زرار r يرجع لحجمه الطبيعي.

when [space] key pressed
  change size by 10

when [r] key pressed
  set size to 100 %

🌟 تحدي إضافي (لو عايز تحدي أكبر!)

  • 🎵 خلّي القط يقول جملة عشوائية كل شوية باستخدام pick random.
  • 🌳 غيّر خلفية المسرح (backdrop) لمكان حلو.
  • 🔢 ضيف متغير "عدد النقرات" يزيد كل ما تدوس على القط (تشويق لـ Session 9!).

⭐ قيّم مشروعك بنفسك (دوس على النجوم)

بعد ما تخلّص، ادّي لنفسك نجوم على كل مهمة عملتها صح. هدفك تجمّع أكبر عدد نجوم! 🌟

1️⃣ القط رحّب بيك باسمك (ask + join)⭐⭐
2️⃣ القط بيتحرك ويبدّل شكله ويرتد (forever + bounce + next costume)⭐⭐
3️⃣ القط بيعمل صوت لما تدوس عليه⭐⭐
4️⃣ القط بياخد قرار بـ if / else⭐⭐
5️⃣ الكيبورد بيكبّر ويصغّر القط⭐⭐
مجموع نجومك: 0 من 10 ⭐

✅ قبل ما تسلّم — اتأكد من ده

  • المشروع بيشتغل لما أضغط العلم الأخضر 🚩
  • القط رحّب بيّ باسمي
  • القط بيتحرك ويبدّل شكله (أنيميشن)
  • فيه صوت بيشتغل لما أدوس على القط
  • فيه سؤال بـ if/else والقط بيرد حسب إجابتي
